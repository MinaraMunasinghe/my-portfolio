# Personal Portfolio Website

Welcome to my personal portfolio repository! This is a clean, responsive digital dashboard designed to showcase my academic projects, software engineering skills, and professional journey.

🌐 **Live Demo:** [https://minaramunasinghe.github.io/my-portfolio/](https://minaramunasinghe.github.io/my-portfolio/)

---

## 🛠️ Portfolio Tech Stack

This website was built completely from scratch using clean frontend web technologies optimized for fast loading and cross-browser compatibility:

* **Structure:** HTML5
* **Styling:** CSS3 (Custom responsive layouts and modal animations)
* **Interactivity:** JavaScript (ES6+)
* **Hosting:** GitHub Pages

---

## 🚀 What's Inside?

When you launch the live site, you can interactively explore:
* **Project Showcases:** Detailed cards featuring full-stack and mobile applications (including TutorSphere and Evoria).
* **Skills Dashboard:** A visual breakdown of my technical competencies.
* **Academic Journey:** A summary of my certifications, ongoing IT studies, and tech focus areas.
